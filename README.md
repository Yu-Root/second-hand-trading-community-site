# 二手物品交易与互换社区平台

一个纯前端、无后端的二手物品交易社区平台，类似校园跳蚤市场或社区闲置交换平台。

## 🛠 技术栈

- **Vue 3** (Composition API + `<script setup>`)
- **Vite** - 快速构建工具
- **Element Plus** - 美观的UI组件库
- **纯前端实现** - 无需任何后端服务器

## 📁 项目结构

```
second-hand-trading-community-site/
├── index.html              # 入口HTML文件
├── package.json            # 项目依赖配置
├── vite.config.js          # Vite配置
├── README.md               # 项目说明文档
└── src/
    ├── main.js             # Vue应用入口
    ├── App.vue             # 根组件（状态管理）
    └── components/
        ├── ItemList.vue        # 物品广场/列表组件
        ├── PublishModal.vue     # 发布物品弹窗组件
        └── UserProfile.vue     # 个人中心组件
```

## ✨ 核心功能

### 1. 首页/物品广场
- 展示卡片式布局的闲置物品列表
- 物品卡片包含：图片、标题、新旧程度、主人昵称
- 标签区分"直接出售"和"以物易物"
- 支持分类筛选：数码 / 图书 / 生活用品

### 2. 发布功能
- 弹出式表单界面
- 填写物品名称、描述、选择分类和新旧程度
- 选择交易类型（出售/交换）
- 提供图片选择或自定义链接
- 发布后实时添加到首页列表

### 3. 交换/交易互动
- 物品卡片上的"我想要购买"/"发起交换"按钮
- 弹出模态框，输入留言信息
- 提交后模拟发送成功提示

### 4. 个人中心
- 展示模拟登录用户的头像和信息
- "我发布的"列表管理自己的物品
- "收到的请求"列表查看别人发给你的交换请求
- "我发出的"列表查看你发起的请求

## 🚀 快速开始

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **构建生产版本**
```bash
npm run build
```

## 📝 说明

- 所有数据都在前端通过 `ref`/`reactive` 维护
- 页面刷新后数据重置为初始Mock数据
- 无需配置任何后端或数据库
- 界面采用清新友好的紫色渐变主题风格
